# 📖 프로젝트 소개
in경산은 단순한 장소 중심의 여행이 아닌, '사람(Host)'을 중심으로 하는 새로운 로컬 관광 서비스를 제안합니다. 대학 도시인 경산의 특성을 살려 로컬 경험과 관계를 중시하는 대학생 및 관광객에게 현지의 장인, 청년 사업가, 골목 상인, 예술가들과의 만남을 연결합니다.

# ✨ 주요 기능
<img width="1265" height="714" alt="Image" src="https://github.com/user-attachments/assets/2e298606-8319-4756-9ec3-e20b3ee792c3" />

### 사람 중심의 탐색, 캐릭터 맵 & 필터링 (Character Map)
단순한 장소 검색을 넘어, '누구를 만나러 갈까?'라는 질문에서 시작하는 새로운 탐색 경험을 제공합니다.

- 호스트 카테고리 필터링: 장인/명인, 청년 사업가, 골목 상인, 예술가 등 사용자가 원하는 유형의 인물을 쉽게 발견할 수 있습니다.

- 위치 기반 캐릭터 맵: 관광지 중심이 아닌, 로컬 호스트의 위치를 중심으로 지도를 표시하여 더욱 객관적이고 현장감 있는 탐색이 가능합니다.

<img width="1265" height="714" alt="Image" src="https://github.com/user-attachments/assets/e40c29ee-f026-4785-b9b0-96f83c761962" />

### 상황 맞춤형 AI 추천 서비스 (AI Recommendation)
사용자의 현재 기분과 상황을 이해하여 최적의 로컬 경험을 제안합니다.

- 문장 분석 추천: "조용한 곳에서 힐링하고 싶어", "시험 끝나고 친구랑 즐길 체험" 등 사용자가 입력한 문장을 AI가 분석하여 실시간으로 호스트와 프로그램을 추천합니다.

- 키워드 기반 매칭: '누구와', '무엇을', '어떻게' 할지에 대한 키워드를 바탕으로 개인화된 탐색 결과를 제공합니다.

<img width="1269" height="710" alt="Image" src="https://github.com/user-attachments/assets/bd2330f4-bbde-4b6b-8cf5-9021fae151b8" />

### 로컬 체험 예약 및 관리 (Booking System)
로컬에서 즐길 수 있는 다양한 프로그램을 손쉽게 예약하고 관리합니다.

- 실시간 예약 및 캘린더: 프로그램별 가능한 날짜와 시간을 확인하고 즉시 예약할 수 있는 시스템을 갖추고 있습니다.

- 상세 프로그램 정보: 소요 시간, 이용 가능 연령, 가격 정보 및 기존 이용객들의 생생한 리뷰를 통해 신뢰도 높은 선택을 돕습니다.

# 📌 컨벤션
### 📚 커밋 메시지

| message | description                              |
| ------- |------------------------------------------|
| feat    | 📁 새로운 기능 추가, 기존 기능을 요구 사항에 맞추어 수정       |
| fix     | 🐛 기능에 대한 버그 수정                          |
| docs    | 📝 문서(주석) 수정                             |
| refactor| ✏️ 기능 변화가 아닌 코드 리팩터링                     |
| test    | ⚙️ 테스트 코드 추가/수정                          |
| chore   | 🛠️ 패키지 매니저 수정, 그 외 기타 수정 ex) .gitignore |

💡 **커밋 메시지 예시**
- feat : 로그인 폼 완성
- fix : 회원가입 에러 해결
- docs : 회원가입 로직 주석 수정
