# 📖 프로젝트 소개
in경산은 단순한 장소 중심의 여행이 아닌, **'사람(Host)'**을 중심으로 하는 새로운 로컬 관광 서비스를 제안합니다. 대학 도시인 경산의 특성을 살려 로컬 경험과 관계를 중시하는 대학생 및 관광객에게 현지의 장인, 청년 사업가, 골목 상인, 예술가들과의 만남을 연결합니다.

# 📌 컨벤션
### 📚 커밋 메시지

| message | description                              |
| ------- |------------------------------------------|
| feat    | 📁 새로운 기능 추가, 기존 기능을 요구 사항에 맞추어 수정       |
| fix     | 🐛 기능에 대한 버그 수정                          |
| docs    | 📝 문서(주석) 수정                             |
| refactor| ✏️ 기능 변화가 아닌 코드 리팩터링                     |
| test    | ⚙️ 테스트 코드 추가/수정                          |
| chore   | 🛠️ 패키지 매니저 수정, 그 외 기타 수정 ex) .gitignore |

💡 **커밋 메시지 예시**
- feat : 로그인 폼 완성
- fix : 회원가입 에러 해결
- docs : 회원가입 로직 주석 수정
